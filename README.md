# Personal Information Manager

## Project Description
This is my first Python project. It stores and displays personal information.

## What I Learned
- Variables and data types
- User input and output
- String formatting with f-strings
- Input validation
- Basic calculations

## How to Run
1. Install Python
2. Open terminal
3. Navigate to folder
4. Run: python personal_info.py
